# 1DCellularAutomata
1次元セルオートマトンを表示する

## コマンド
- /position1 - 範囲指定pos1
- /position2 - 範囲指定pos2

-/gen1dautomata  rule_num ゲームルール番号を指定して描画

### 最初の1列が初期値
- 黒羊毛→1
- 白羊毛→0
